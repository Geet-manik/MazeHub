ROUND 1
You have been trapped in a room and the door has a password lock on it. The password is of 9 digits, which contain each of the digit from 1 to 9. The number is divisible by 9 when the number of digits is 9. When we remove the last digit the number of digits become 8 and the number is divisible by 8 and so on. There is increasing poisonous gas in the room. Can you find out the password to save yourself?

Divisibility rule for:

  • 2 : Last digit should be even
  • 3 : sum of digits should be divisible by 3
  • 4: Last 2 digits must be divisible by 4
  • 5: last digit should be 0 or 5
  • 6. divisible by both 2 and 3
  • 7: The difference between twice the unit digit of the number and the remaining part of the number should be divisible by 7 or should be 0
  • 8: last 3 digits should be divisible by 8
